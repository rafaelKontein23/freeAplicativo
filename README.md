AppFreela
Conectando Freelancers a Oportunidades
Este projeto é um aplicativo desenvolvido para conectar freelancers de diversas áreas a pessoas que buscam seus serviços. A plataforma permite que profissionais encontrem trabalhos e que clientes encontrem freelancers qualificados de forma prática e eficiente.

📌 Funcionalidades
✅ Cadastro e perfil para freelancers e contratantes
✅ Busca de oportunidades e serviços disponíveis
✅ Sistema de propostas e negociação direta
✅ Feed com avaliações e recomendações

🛠 Tecnologias Utilizadas
O aplicativo foi desenvolvido utilizando as seguintes tecnologias:

XML & Jetpack Compose – Interface moderna e dinâmica
SQLite – Banco de dados local para armazenamento eficiente
Retrofit – Comunicação com APIs de forma rápida e segura
Coroutines – Execução assíncrona otimizada
Kotlin – Linguagem principal para desenvolvimento Android
À medida que novas tecnologias forem integradas, esta lista será atualizada! 🚀
